◆ TL;DR

━━━━━━━━━━━━━━━━━━━━

普林斯顿团队在 ICLR 2026 发表了一篇论文，用信息瓶颈（Information Bottleneck）理论给 LLM 训练过程装了个仪表盘。核心结论：训练分"先胖后瘦"两阶段，压缩效率（而非记住了多少）预测下游性能，后训练偏好信号与性能的相关系数高达 0.76。

论文：Henry C. Conklin et al., "Learning is Forgetting: LLM Training As Lossy Compression", ICLR 2026. arXiv: 2604.07569

如果你读过 69 期（#）和 99 期（#），你已经亲眼看到过这些现象——PCA 维度骤降、拓扑夺舍、WD 逼出压缩。普林斯顿这篇论文做的事情是：把你在显微镜下看到的微观结构变化，接上了一个信息论刻度的仪表。

━━━━━━━━━━━━━━━━━━━━

◆ 我们早就知道训练是压缩——但缺一把尺子

━━━━━━━━━━━━━━━━━━━━

快速回忆。

69 期（#）我们用 2 层 128 维 Transformer 学模运算，发现训练是两阶段的：先"局部流形发现"，再"全局粘合"。PCA 维度从 78 骤降到 8——表征空间被暴力压缩了一个数量级。模乘法只完成第一阶段就停了，学到商群 Z_12 而非完整乘法群，纯度 99.4%。Grokking 是概率性相变，成功率 67%。最扎心的发现是"能表示不等于能发现"——数学上 1 维就够编码，但模型需要 16 维的工作空间才能摸到那个结构。

99 期（#）把压缩的残酷推到了极致。"拓扑夺舍"——模型全程测试准确率 100%，但内部表征被彻底重组。外层 stride=1 编码崩溃，内层 stride=4 接管。WD 是压缩的发动机：先逼出结构，再摧毁结构（1M 步 100% → 5M 步 73%）。容量和压力必须匹配——大模型配小 WD 什么都学不到。模型自己算出 gcd(12,8)=4 作为最优编码基。

127 期（#）我们在 Mamba-3 上看到了推理时的遗忘——SSM 的 A 矩阵就是一条遗忘曲线，"压缩必然有损"是信息论铁律。

三期加在一起，我们已经建立了这些直觉：

- 训练有两个阶段：先学再压
- 压缩是暴力的：旧编码会被拆掉，新编码会接管
- 容量和压力要匹配：太大或太小都不行
- 遗忘不是 bug，是 feature

**但我们一直缺一个东西：一把统一的尺子。**

69 期看的是 PCA 维度和 Betti 数。99 期看的是邻接得分和 stride 纯度。127 期看的是 A 矩阵的衰减速率。每个实验各有各的度量，无法横向比较。你没法说"69 期的 Grokking 模型压缩效率是 0.73，99 期的嵌套模型压缩效率是 0.81"——因为这把尺子根本不存在。

普林斯顿这篇论文给了这把尺子。

━━━━━━━━━━━━━━━━━━━━

◆ 信息瓶颈 + 球面软熵：一套能在万维空间里跑的仪表

━━━━━━━━━━━━━━━━━━━━

信息瓶颈（Information Bottleneck, IB）理论的目标函数：

F_beta = I(X;Z) - beta * I(Y;Z)

X 是输入 token，Z 是模型隐层表征，Y 是下一个 token。I(A;B) 是互信息——知道 A 之后你对 B 的不确定性减少了多少，概率论课上学过 H(X) 的话：I(A;B) = H(A) + H(B) - H(A,B)。

两项拆开看：

**I(X;Z) = 复杂度。** 表征 Z 保留了多少原始输入 X 的信息。越小越好——说明冗余被扔掉了。

**I(Y;Z) = 表达力。** 表征 Z 能预测多少目标 Y 的信息。越大越好——说明有用的留下来了。

**beta** 控制权衡方向。beta 大偏向保信息，beta 小偏向压体积。

把 beta 从 0 扫到正无穷，每个值对应一个最优压缩策略，连起来就是 IB 边界——帕累托前沿。在边界上的模型是理论最优，低于边界的还有改进空间。

关键指标——**最优度（Optimality）**：

Optimality = I(Y;Z) / I(X;Z)

直觉：你花 1 bit 复杂度换回了多少 bit 表达力。越接近 1，每一 bit 存储都没浪费。

理论到此为止，硬骨头在度量。LLM 的隐层是 4096 维（7B）到 5120 维（32B）的连续向量。要在这种空间里估计概率分布，传统分箱法需要 2^4096 个格子——比宇宙原子数多了不知道多少个数量级。这就是维度灾难。

普林斯顿的方案叫**球面软熵估计器**（Spherical Soft Entropy Estimator），五步：

1. L2 归一化，把所有隐层向量映射到单位超球面，只保留方向信息
2. 在球面上均匀采样 n=8192 个参考点，当"软格子"中心
3. 对每个 embedding 算和所有参考点的余弦相似度，过 softmax 得到软分配概率
4. 汇总所有点的软分配，算 Shannon 熵 H(Z)
5. 算条件熵和互信息：I(X;Z) = H(Z) - sum_x P(x)*H(Z|X=x)

**为什么必须"软"？** 高维空间有个反直觉的性质叫"橘子皮定理"——d 维球体当 d 很大时，几乎所有体积集中在表面薄壳里，内部是空的。这意味着数据点全挤在某些低维流形上，硬边界切出来的格子绝大多数是空的。更要命的是，硬边界左边 0.001 和右边 0.001 的两个点被分到不同格子，但它们实际上几乎一样。软分配用连续概率代替 0/1 分类，消除了这种人为噪声。

作者在合成数据（已知真实互信息值）上验证了精度。这是后面所有实验的地基。

◆ 先胖后瘦——和我们在 69/99 期看到的是同一个过程

━━━━━━━━━━━━━━━━━━━━

拿着这把尺子去量 OLMo2（AI2 开源的大模型，1B/7B/32B 三个规模，每个规模几十个训练中间 checkpoint），结果非常漂亮。

**Phase 1——拟合期。** I(Y;Z) 快速上升，I(X;Z) 也在涨或基本不动。模型在疯狂吸收——不管有用没用，先塞进来。IB 平面上的轨迹向右上移动：复杂度增加，表达力也增加。

**Phase 2——压缩期。** I(Y;Z) 饱和不再上升，I(X;Z) 开始下降。模型开始扔垃圾——把对预测无帮助的冗余从表征里挤出去。轨迹向左移动：复杂度降低，表达力不变。

转折点精确对应 next-token loss 饱和时刻。loss 不再下降时，模型不是"学完了"，是"开始学遗忘了"。

现在把这个结果和我们自己的实验对齐：

**Phase 1 = 69 期的"局部流形发现" = 99 期的外层拓扑涌现。** 这个阶段模型在高维空间里到处撒网，PCA 维度还很高（69 期的 78 维），各种编码方案并存（99 期的 stride=1 和 stride=4 共生）。信息论视角就是 I(X;Z) 在涨——模型把输入信息往表征里灌，不做筛选。

**Phase 2 = 69 期的"全局粘合" = 99 期的"拓扑夺舍"。** PCA 维度从 78 骤降到 8——这就是 I(X;Z) 下降的微观实现。99 期里外层 stride=1 崩溃、内层 stride=4 接管——旧编码被拆掉、新编码接管，就是"扔垃圾"的具体动作。WD 推着 I(X;Z) 往下走，逼模型用更少的 bit 编码同样的有用信息。

**1B 振荡失败 = 99 期的"容量/压力不匹配"。** 1B 模型在 Phase 2 出现明显振荡，最终没能压好，轨迹离 IB 边界有明显距离。99 期我们看到的是：大模型配小 WD 什么都学不到——容量太大压力太小，模型没有动力压缩。反过来，容量太小压力正常（1B 的情况），模型想压但没有余量做精细操作——删掉 A 的冗余会破坏 B 的有用信息，顾此失彼。

7B 和 32B 成功逼近 IB 边界——它们的压缩接近理论最优。

**一句话：69/99 期是显微镜看细胞分裂，普林斯顿是 CT 扫描看器官轮廓。微观拓扑视角和宏观信息论视角，看的是同一个"先胖后瘦"的过程。**

这也给了 scaling law 一个信息论侧面的解释：大模型不是因为"记住了更多知识"而更强，是因为有更大的参数空间来执行精细压缩——它们忘得更聪明。

◆ 压缩质量预测性能——后训练是在调 beta

━━━━━━━━━━━━━━━━━━━━

单模型轨迹好看，横向验证更硬。47 个开源模型，6 个家族（OLMo、Llama、Qwen、Mistral 等），6 个主流 benchmark。算 Spearman 秩相关系数：

**复杂度 I(X;Z)：r = -0.38（p=0.006）。** 负相关。记住的原始信息越多，性能越差。冗余信息是负债。

**表达力 I(Y;Z)：r = 0.08（不显著）。** 光有信息量没用。I(Y;Z) 高可能只是因为 I(X;Z) 也高——有用信息被冗余淹没了。

**最优度 Optimality：r = 0.52（p<0.001）。** 压缩效率才是关键。不是"记得多"就好，是"记对了"才好。

**偏好信息 I(Z;preference)：r = 0.76（p<0.001）。** 这是最炸的数字。

r = 0.76 意味着什么？模型表征中编码了多少人类偏好信息，比其他所有指标都更能预测最终性能。后训练（RLHF、DPO）灌进去的信息含量，是下游表现的最强预测因子。

现在回到 99 期。我们在那篇文章里看到：WD 的大小决定了模型最终选择哪种编码——stride=1 还是 stride=2 还是 stride=4。WD 不是在教模型"答案是什么"，是在施加压力，逼模型在有限容量里选择一种编码方案。小 WD 压力不够，模型赖在 stride=1 的冗余编码里不动；大 WD 压力过猛，好的编码也被摧毁。

**后训练的偏好信号干的是同一件事，但更精准。** WD 是无差别压力——对所有方向施加同样的惩罚。RLHF/DPO 是定向压力——告诉模型"这个方向的信息值钱，那个方向的不值钱"。用 IB 理论的语言说：后训练不是在教新知识，是在调 beta 的方向——把人类觉得重要的信息权重调高，把无关的压下去。

这就解释了 r = 0.76 为什么这么高：偏好对齐直接操控了信息瓶颈的压缩方向。

论文还发现了一个规模效应：大模型把更多表征空间分给了上下文信息（bigram、trigram、4-gram 级别的条件互信息 I(Z; bigram | token) 等），而非单 token 级统计。大模型更擅长编码"词与词之间的关系"而非"某个词本身的频次"——和人读文章记住的是句意而非字频一个道理。

作者还建议 Optimality 可以当训练停止标准——比跑完整 benchmark 便宜得多，只需要在隐层采样算互信息。如果 Optimality 开始饱和甚至下降，就可以考虑停了。

◆ 三层遗忘：完整的图景

━━━━━━━━━━━━━━━━━━━━

现在可以把三条线串起来了。

**第一层：训练时的信息压缩（普林斯顿，宏观）。** IB 理论视角。Phase 1 吸收，Phase 2 压缩。I(X;Z) 下降就是遗忘。这是最宏观的描述——像 CT 扫描，看到器官在缩小，但不知道里面哪些细胞在死。

**第二层：训练时的拓扑重组（69/99 期，微观）。** 几何/拓扑视角。PCA 维度骤降、Betti 数变化、stride 编码切换。这是 I(X;Z) 下降的微观实现——"扔垃圾"不是均匀地扔，是旧的编码方案整体崩溃，新的更高效的编码方案接管。拓扑夺舍就是有损压缩的具体执行方式。

**第三层：推理时的遗忘（127 期，实时）。** SSM 的 A 矩阵 = 遗忘曲线。每一步推理都在对历史信息做指数衰减。这不是训练阶段的压缩，是推理阶段的实时丢弃。训练教会了模型"该记什么该忘什么"的策略，推理时模型按这个策略一步步执行遗忘。

三层叠在一起：训练时先用信息压缩（第一层）选出最优编码方案，具体执行方式是拓扑重组（第二层）；训练完成后，模型带着这套压缩策略上线，推理时按策略实时遗忘（第三层）。

这才是完整的"学习即遗忘"。

不是一个比喻，是三个不同尺度上的同一个信息论过程：用有限的表征容量，对无限的输入分布做有损压缩。训练时压一次（学会策略），推理时压一次（执行策略）。从头到尾，模型一直在扔东西。

它学会的不是记住什么，是忘掉什么。
